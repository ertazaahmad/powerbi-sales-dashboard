# Power BI Sales Dashboard

This project analyzes sales data using Power BI.

## Tools Used
- Power BI
- SQL
- Excel
- DAX

## Insights
- Total Sales by Region
- Profit Trends
- Monthly / Daily Sales Trend
- Total Sales By Category
